Chi Tiết Dự Án: Hệ thống Quản lý Thư viện

Tổng quan

Dự án Full-Stack nhằm tự động hóa quy trình quản lý tại thư viện. Mục tiêu là cung cấp một nền tảng thân thiện, hiệu quả cho cả người dùng (tra cứu sách) và quản trị viên (quản lý kho và độc giả).

Vai trò cá nhân

const Role = 'Full-Stack Developer';

  • Thiết kế wireframe và mockup giao diện.
  • Phát triển Front-end (HTML, CSS, JavaScript) cho các trang Tra cứu và Dashboard.
  • Thiết kế và triển khai CSDL (MySQL).
  • Lập trình Back-end (PHP) để xử lý logic kinh doanh, API mượn/trả.
  • Kiểm thử tích hợp hệ thống.

Chức năng chính

// Các modules đã được phát triển:

  • Đăng nhập/đăng ký với mã hóa mật khẩu.
  • Tra cứu sách nâng cao (tìm kiếm theo từ khóa, lọc theo thể loại, tác giả).
  • Quản lý danh mục sách (CRUD).
  • Quản lý mượn/trả sách, hiển thị thông báo quá hạn.
  • Phân quyền người dùng (Admin, Thủ thư, Độc giả).

Công nghệ sử dụng

  • HTML5
  • CSS3
  • JavaScript
  • PHP (Back-end Logic)
  • MySQL (Database)
  • Deployment (XAMPP/Localhost)
