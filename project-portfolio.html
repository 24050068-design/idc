Chi Tiết Dự Án: Website Cá Nhân (Portfolio)

Tổng quan

Dự án này là một trang web tĩnh (Portfolio) nhằm mục đích giới thiệu hồ sơ nghề nghiệp cá nhân, bao gồm các phần: Giới thiệu, Kỹ năng, Dự án, và Liên hệ. Mục tiêu chính là trình bày năng lực lập trình Front-end và gu thẩm mỹ thiết kế.

Vai trò cá nhân

const Role = 'Front-End Developer / UI Designer';

  • Thiết kế bố cục giao diện và phối màu để tạo dấu ấn cá nhân.
  • Sử dụng CSS để tạo các hiệu ứng chuyển động và Animation.
  • Đảm bảo tính Responsive, tối ưu hiển thị trên mọi thiết bị (điện thoại, tablet, desktop).
  • Tối ưu hóa hiệu suất tải trang cho trải nghiệm người dùng tốt nhất.

Cấu trúc trang

// Các modules chính:

  • Phần Giới thiệu (Hero Section).
  • Danh sách Kỹ năng (có Progress Bar/Biểu đồ).
  • Thư viện Dự án (Gallery).
  • Form/Thông tin Liên hệ.

Công nghệ sử dụng

  • HTML5 (Cấu trúc)
  • CSS3 (Bố cục Flexbox/Grid, Animation)
  • JavaScript (Tương tác nhỏ)
